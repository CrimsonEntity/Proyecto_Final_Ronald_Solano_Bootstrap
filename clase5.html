Imagenes

Bootstrap permite optar por imagenes que tengan un comportamiento responsive y que cambian según el dispositivo por el cual se accede a la web.

Imagenes responsive.

Bootstrap puede hacer que la imagen ingresada abarque el 100% del ancho de la página y un alto autómatico agregando la clase .img-fluid.

Imagenes de miniaturas.

Además de la las utilidades de border se puede usar la clase .img-thumbnail para darla a una imagen una apariencia de un borde redondeado de 1px.

Alineamiento de Imagenes.

Se pueden alinear imagenes con la ayuda de los flotantes utilizando las clases .rounded float-start, .rounded float-end, .rounded mx-auto d-block y con la clase .text-center.

Figuras

Ejemplos de como mostrar imagenes y texto relacionados con el componente de figura de bootstrap.

Cada vez que se necesita mostrat contenido como una imagen con un subtítulo opcional se debe considerar usar la clase .figure.

La clase .figure tiene incluidos .figure-img y .figure-caption clases que facilitaran la labor.
